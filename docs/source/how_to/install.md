# Install OKPaletteLab

OKPaletteLab is available on PyPI. You can install it using pip:

```bash
pip install ok-palette-lab
```

## Optional Dependencies

OKPaletteLab has optional dependencies for specific visualization libraries.
You can install them by specifying the desired extras or by installing them yourself.

### Plotly Support

To use OKPaletteLab with Plotly:

```bash
pip install ok-palette-lab[plotly]
```

### Matplotlib Support

To use OKPaletteLab with matplotlib:

```bash
pip install ok-palette-lab[matplotlib]
```
